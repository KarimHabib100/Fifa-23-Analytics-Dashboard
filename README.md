# FIFA 23 Players Analytics Dashboard

An interactive Plotly Dash dashboard built for the **Data Visualization Final Project** (Project 8).
It analyzes football player statistics from the FIFA 23 video game dataset, delivering a
scouting-style analytics experience across clubs, leagues, nationalities, and FIFA editions.

---

## Dataset

| Field | Detail |
|-------|--------|
| **Name** | FIFA 23 Complete Player Dataset |
| **Source** | https://www.kaggle.com/datasets/stefanoleone992/fifa-23-complete-player-dataset |
| **Raw file** | `male_players.csv` |
| **Scope** | ~18,000 male players across FIFA editions 15–23 |

### Columns Selected and Why

| Column | Purpose |
|--------|---------|
| `short_name`, `long_name` | Player identification and hover labels |
| `player_positions` | Source for the engineered `position_group` column |
| `overall` | Primary rating metric — used in all comparison and distribution charts |
| `potential` | Secondary rating metric — violin chart, bubble size, metric toggle |
| `value_eur` → `market_value_in_eur` | Bubble chart Y-axis (capped at 99th percentile) |
| `wage_eur` → `wage_in_eur` | Box chart Y-axis (weekly wage distribution) |
| `age` | Scatter/bubble X-axis; line/area pseudo-time axis |
| `height_cm`, `weight_kg` | Supporting physical attributes |
| `league_name` | Stacked/clustered grouping; league dropdown filter |
| `club_name` | Column chart categories (top clubs) |
| `nationality_name` | Bar chart categories (top nationalities) |
| `preferred_foot` | Violin chart X-axis (Left vs Right distribution) |
| `work_rate` | Supplementary player attribute |
| `pace`, `shooting`, `passing`, `dribbling`, `defending`, `physic` | Six FIFA skill pillars |
| `fifa_version` | FIFA edition identifier (reserved for edition-based analysis) |
| `position_group` *(engineered)* | GK / DEF / MID / FWD — used across all charts |

## Chart Types and Analytical Questions

| Week | Chart Type | Analytical Question Answered |
|------|-----------|------------------------------|
| 1 | **Column Chart** | Which club has the highest average player rating? |
| 1 | **Bar Chart** | Which nationalities produce the highest-rated players? |
| 2 | **Stacked Column Chart** | How is each position group composed by preferred foot (Left vs Right)? |
| 2 | **Stacked Bar Chart** | What is the player count composition by position group across top leagues? |
| 2 | **Clustered Column Chart** | Which position group has the largest gap between Overall and Potential rating? |
| 2 | **Clustered Bar Chart** | Which league pays the highest average wages, and how does this differ by position? |
| 3 | **Scatter Chart** | Does age influence a player's overall rating? Are there outliers or clusters? |
| 4 | **Bubble Chart** | How does age relate to market value, and does potential predict market value? |
| 5 | **Histogram** | How are overall ratings distributed across position groups? |
| 6 | **Box Chart** | How are weekly wages distributed across position groups (five-number summary)? |
| 7 | **Violin Chart** | Does preferred foot affect a player's potential distribution (KDE silhouette)? |
| 8 | **Line Chart** | How does average rating change across age groups per position? |
| 9 | **Area Chart** | How does the cumulative player count build up across career ages? |

## Interactive Controls

| Control | Type | Affects |
|---------|------|---------|
| **League** | Dropdown | Column chart, Bar chart, Stacked charts, Clustered charts, Line chart, Area chart |
| **Position Group** | Dropdown | Stacked charts, Clustered charts, Scatter, Bubble, Histogram, Box, Violin |
| **Age Range** | Range Slider | Scatter, Bubble, Histogram, Box, Violin, Column chart, Bar chart |
| **Metric** | Radio Items (Overall / Potential) | Column chart, Bar chart, Line chart |

## How to Run

**Step 1 — Install dependencies:**
```bash
pip install -r requirements.txt
```

**Step 2 — Prepare the data:**

Download `male_players.csv` from the Kaggle link above and place it inside `data/`.
Then open and run `notebooks/preprocessing.ipynb` top-to-bottom to generate `cleaned_players.csv`.

**Step 3 — Launch the dashboard:**
```bash
cd Team_FIFA23_Analysis
python app.py
```

Open your browser at: **http://127.0.0.1:8050/**

> **Note:** Use Python 3.11 if you have multiple Python versions installed.
> Confirm with: `python --version`
